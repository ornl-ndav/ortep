ORTEP-III Updates

Version 1.0.2 - November 4, 1996

  • Corrects an error related to the use of "features" with the 105, 106, 405, 415, 406, and 416 instructions.

  • Corrects an error related to the logic used when more than one vector screening condition is provided with the 100 and 400 series instructions.

Download the Fortran source code or compiled executables of Version 1.0.2.

NOTE: ALL the changes in Version 1.0.2 of ORTEP-III are in SUBROUTINE SEARC. If you already have the source code for Version 1.0.1, make the following THREE changes in SUBROUTINE SEARC:

CHANGE   TO
inum=0
DO 153 KI=ITAR1,ITAR2		    DO 153 KI=ITAR1,ITAR2
inum=0
inum=0
DO 396 KI=ITAR1,ITAR2		    DO 396 KI=ITAR1,ITAR2
inum=0
logc=ain(8)    logc=ain(9)

Version 1.0.1 - October 1, 1996


  • Adds dashed bonds (for hydrogen bonds)

  • Adds line bonds without atom symbols (for wireframe structures)

Read the README file for Version 1.0.1.

Read the updated online documentation for the 800 instructions.

NOTE: ALL the changes in Version 1.0.1 of ORTEP-III are in SUBROUTINE BOND. If you already have the source code for Version 1.0, download the new SUBROUTINE BOND code and substitute it for the original code.
