Crystallographic Data Input

Instructions   View summary table   Download summary table (Postscript, 90K)

    Format

    100 - Structure Analysis

      101 and 102: Distances and Angles within Sphere of Enclosure
      103: Principal Axes of Thermal Motion
      105 and 106: Distances within Convoluting Sphere of Enclosure or Reiterative Convoluting Sphere of Enclosure

    200 - Plotter Control

    300 - Drawing Parameters

      301: Drawing Boundaries, Margin, and Viewing Distance
      302: Title Rotation
      303: Retrace Displacement
      304: Ellipse Smoothness

    400 - ATOMS Array

      401/411: Atom Designator Run Add/Eliminate
      402/412: Sphere of Enclosure Add/Eliminate
      403/413 and 404/414: Box of Enclosure Add/Eliminate and Triclinic Box of Enclosure Add/Eliminate
      405/415 and 406/416: Convoluting Sphere of Enclosure Add/Eliminate and Reiterative Convoluting Sphere of Enclosure Add/Eliminate
      410: Clear ATOMS Array

    500 - Orientation

      501: Explicit Reference Cartesian System Assignment
      502: Reference Cartesian System Rotation
      503: Working Cartesian System Rotation (Stereoscopic Rotation)
      504: Reference Cartesian System Origin Translation
      505: Reference Cartesian System Origin at Centroid
      506: Reference Cartesian System Origin at Centroid and Inertial Axis

    600 - Positioning and Scaling

      60x: Normal Modes of Positioning and Scaling
      61x: Incremental Modes of Positioning and Scaling

    700 - Atom Plotting

    800 - Bond Plotting

      801/811: Explicit Bonds
      802/812 and 803/813: Implicit Bonds
      821 and 822: Bond Overlap

    900 - Label Plotting

    1100 - Save Sequence

    1001, 821, 822 - Overlap Correction

      1001: Projected Outline Storage
      821: Explicit Bond Outline Storage
      822: Implicit Bond Outline Storage

    -1 and -2 - Job Termination and Restart
